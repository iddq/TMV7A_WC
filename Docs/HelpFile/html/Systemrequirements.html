System requirements

Getting Started ››


The TMV7A application has been tested on MS WIndowsXP, MS Vista Business and MS Windows8 and requires a single RS-232 serial port. It has been tested with both on-board serial ports and USB serial cables.


A printer is required in order to print reports from the application.
